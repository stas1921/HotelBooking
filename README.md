## 🎯 Описание проекта

**HotelBooking** — это онлайн-платформа, объединяющая путешественников и владельцев отелей. Приложение позволяет:

- Выбирать номера по категориям: Эконом, Комфорт, Люкс, Президентский
- Бронировать номера с выбором дат и расчётом стоимости
- Просматривать фото отелей и адреса на Яндекс Картах

---

## ⚡ Функциональные возможности

### 👤 Пользователи
- Регистрация и авторизация (Spring Security)
- Ролевая модель: `TOURIST`, `HOTEL_MANAGER`, `ADMIN`
- Хранение паролей в зашифрованном виде (BCrypt)

### 🏨 Отели
- Просмотр списка отелей с фото и описанием
- Добавление, редактирование, удаление отелей (для администратора)
- Привязка к городу, адресу и звёздности

### 🛏️ Номера
- 4 категории: Эконом, Комфорт, Люкс, Президентский
- Разная цена и вместимость для каждой категории
- Визуальное отображение всех категорий в карточке отеля

### 🗺️ Интеграция с картами
- Кликабельный адрес отеля
- Переход на Яндекс Карты с маршрутом

### 📅 Бронирования
- Выбор дат заезда и выезда
- Автоматический расчёт количества ночей
- Расчёт итоговой стоимости
- Подтверждение бронирования

## 🛠 Технологический стек

| Категория | Технологии |
|-----------|------------|
| **Backend** | Java 26, Spring Boot 3.4.0 |
| **База данных** | PostgreSQL 16 |
| **Security** | Spring Security, BCrypt |
| **ORM** | Spring Data JPA, Hibernate |
| **Миграции** | Liquibase |
| **Логирование** | Spring Boot Actuator, SLF4J |
| **Фронтенд** | HTML5, CSS3, JavaScript (Vanilla) |
| **Сборка** | Maven |
| **Тестирование** | JUnit 5, Mockito (в разработке) |
